# practice

準備段階でやったこと<br>
・Node.js のインストール
　（やらなくてよかったかも）
```
sudo apt install nodejs
sudo apt install npm
```
・PostgreSQL クライアントのインストール<br>
　Node.js 用の PostgreSQL クライアントライブラリをインストールします。
```
npm install pg
```
・Express のインストール
　Express は Node.js 用の軽量な Web アプリケーションフレームワークです。
```
npm install express
```
